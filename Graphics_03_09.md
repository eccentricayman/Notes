# Parametric Equations

- A way to describe a curve with respect to an independent variable (t).

  - Non-parametric: y = mx + b

  - Parametric:
    $$
    x = f(t)\\y = g(t)\\\text{Line}\\x = x_0 + (\triangle x)*t\\y=y_0+(\triangle y)*t \\ \\ t: 0 \text{ --> }1
    $$

  - Circle Parametric:

  $$
  x = r*cos(t)+cx\\
  y = r*sin(t)+cy
  $$
