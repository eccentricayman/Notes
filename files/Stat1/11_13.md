## Continuity Corrections

1. $P(x = X) = P(X - .5 \le x \le X + .5)$
2. $P(x \le a) = P(x \lt a + .5)$ 
3. $P(x \ge a) = P(x \ge a -.5)$
4. $P(x \lt a) = P(x \le a - .5)$ 
5. $P(x \gt a) = P(x \gt a + .5)$
